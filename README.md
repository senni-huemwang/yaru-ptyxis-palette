# Yaru Palette for Ptyxis Terminal

Ubuntu's Yaru theme inspired palette for Ptyxis terminal (since Ubuntu's own theme were too purple)

[Screenshot](Screenshot.png)

## Installation

Clone the repository, and move the file into Ptyxis' palettes directory.

```
cp Yaru.palette ~/.local/share/org.gnome.Ptyxis/palettes
```
